# Upgrade Notes v4

- Detail panel now shows:

  - why the topic matters
  - exam question patterns
  - common mistakes
  - memory tips

- Added more hard/trap-style questions across security, database, network, management, strategy, cloud, and AI.

- Added Docker support:
  - Dockerfile

  - nginx.conf

## Why

The goal is to make the app more useful for real IT Passport study, not only a visual graph. The v4 changes make the app more study-oriented and easier to run on any machine.
