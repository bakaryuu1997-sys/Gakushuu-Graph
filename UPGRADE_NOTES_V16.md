# V16 Guided Study + Clear Graph Upgrade

## What changed

1. **Start Here becomes the default view**
   - The app now opens with a clear learning start screen.
   - It tells the learner which node to study first and why.
   - The Study Path is shown as ordered phases, so the learner knows what comes before and after.

2. **Study content sits next to the node list**
   - For learning views such as Start Here, Dashboard, Today, Study Path, Need Review, All Nodes, Visual Maps, and 7/14/30 Plan, the selected lesson appears in a sticky reader panel directly beside the list.
   - The user no longer needs to scroll far up/down to find the lesson after choosing a node.

3. **Graph layout follows Study Path order**
   - Graph nodes are positioned by learning phase from left to right.
   - This makes the map easier to read as a learning flow instead of a scattered graph.

4. **Clearer study workflow**
   - Recommended workflow is now:
     Start Here → read lesson beside list → Category Practice → Japanese Exam → Full Map.

## How to use

Start with the **Start Here** view. Click the recommended node. Read the lesson in the right panel. Mark it as learning/mastered. Continue through the next nodes in order.

Use **Full Map** only when you want to understand relationships visually.
