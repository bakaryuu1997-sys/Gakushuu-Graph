# Upgrade Notes V6 — AI Passport Main Course + Multi-Course Foundation

## What changed

- Added `AI Passport` as the default/main course.
- Kept `IT Passport` as a second selectable course.
- Added a multi-course registry so future courses can reuse the same app shell.
- Added a course switcher inside the Study Navigation panel.
- Progress, favorites, and recent nodes are now saved separately per course.
- Added AI Passport graph data, study path, lessons, and quiz questions.
- Updated header copy from IT Passport-only to a broader Vibe Study Atlas concept.

## New AI Passport content

AI Passport now includes core topic groups:

- AI Basics
- Data and Machine Learning
- Generative AI
- AI Ethics and Law
- AI Business Use

Important lesson nodes include:

- AI Passport
- Machine Learning
- Data Preprocessing
- Generative AI
- Hallucination
- AI Ethics and Law
- AI Project Flow

## Remaining limitations

- AI Passport content is still MVP-level, not a full official exam bank.
- Quiz questions are keyword-focused and practical, but not yet a full 300-question mock test set.
- Compare View exists conceptually in the roadmap but is not yet a full dedicated screen.
- Course data is static TypeScript data. There is no editor/import system yet.
- Bundle warning remains because React Flow + html-to-image + data are bundled together.
