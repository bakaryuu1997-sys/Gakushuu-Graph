# V10 Learning Completeness Upgrade

## What changed

- Added universal lesson fallback for all AI Passport nodes, so every node now has readable Vietnamese/Japanese lesson content.
- Added harder business, ethics, law, governance, AI security, RAG, prompt injection and ML evaluation scenario questions.
- Added a 7 / 14 / 30 day Study Calendar view that arranges nodes in learning order and limits each day to a manageable learning queue.
- Added Japanese Glossary Quiz mode for JP→VI and VI→JP terminology practice.
- Improved the course data registry to include the new lesson and quiz datasets.

## Learning direction

This version focuses on:

1. Learn in order using Study Path or 7/14/30 Plan.
2. Read center lesson content for the selected node.
3. Use Glossary and Glossary Quiz to memorize Japanese terms.
4. Use Answer Review for explanations.
5. Use Japanese Exam Mode for Japanese-only practice.
6. Use Category Practice and hard scenario questions for business/ethics judgment.

## Notes

The questions are original simulated practice questions based on AI Passport topic patterns. They are not copied from official past exam banks.
