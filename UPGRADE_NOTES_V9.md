# V9 Learning Depth Upgrade

This version focuses on study quality instead of tracking complexity.

## Added

- Expert Vietnamese lessons for 50 high-importance AI Passport nodes.
- AI Passport Japanese-Vietnamese glossary.
- Answer Review view to browse all questions by topic with correct answer and explanation.
- Japanese Exam Mode with Japanese-only questions, choices, and explanations.
- More real-life examples shown in the lesson panel when available.
- Extra Japanese-style exam questions with Japanese choices.

## Philosophy

The app should help users learn in this order:

1. Follow Study Path.
2. Read the centered lesson content.
3. Use Glossary for Japanese terms.
4. Practice by category.
5. Use Answer Review to understand answers.
6. Use Japanese Exam Mode to prepare for Japanese wording.

## Not included yet

- Official past-question bank. Questions are original practice items inspired by common AI Passport themes.
- Full expert lessons for all 250+ nodes. This version prioritizes the most important nodes first.
