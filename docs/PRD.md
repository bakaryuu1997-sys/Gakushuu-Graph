# PRD — IT Passport Knowledge Graph

## Vision
A visual learning map that helps Vietnamese learners understand Japanese IT Passport concepts through an interactive graph.

## MVP Scope
1. Show IT Passport concept graph.
2. Click a node to read bilingual explanation.
3. Search and filter nodes.
4. Track learning progress locally.

## Out of Scope
- Backend
- Login
- AI API
- Cloud sync
- Admin editor

## Success Criteria
- User can understand the 3 major IT Passport fields at a glance.
- User can click any node and learn meaning, exam point, and keywords.
- App works locally after `npm install && npm run dev`.
