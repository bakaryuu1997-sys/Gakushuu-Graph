# V14 Study-first + Fullscreen Knowledge Map

## What changed

- Added a dedicated Graph Map view for large, diagram-like exploration.
- Added Fullscreen Map mode with a right-side node drawer.
- Kept lesson reading as the center of the app and moved graph into a supporting role.
- Improved focused graph usability with a clear Full Map button.
- Updated graph positioning to a more readable left-to-right layered map instead of scattered clusters.
- Added graph reading guidance: prerequisite lines, comparison lines, HOT nodes.
- Improved the map inspector so clicking a node keeps context and shows related learning nodes.

## Why

The previous versions had strong content, but the graph and lesson layout could still feel difficult to read. This version separates the app into:

- Study mode: lesson content first.
- Graph mode: large visual map first.
- Fullscreen mode: architecture-diagram style map exploration.
