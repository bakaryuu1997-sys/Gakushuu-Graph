# V8 Deep AI Passport Study Upgrade

## What changed

- AI Passport remains the default/main course.
- Expanded AI Passport nodes to about 250+ nodes.
- Added a higher-quality AI Passport quiz bank with 130 curated questions on top of the existing bank.
- Quiz cards now reveal the correct answer and explanation after the user selects an option.
- Added Category Practice mode for:
  - Generative AI
  - Ethics / Law
  - Data / ML
  - Business Use
- Rebuilt AI Passport Study Path into 10 ordered phases so learners know exactly what to study first.
- Added deeper lesson content for the most important AI Passport topics.
- Renamed Mistake Review behavior toward trap/easy-confusion review; it does not store wrong answers.
- Kept Docker support and production build support.

## Important note

The quiz bank is a high-quality practice bank modeled after common AI Passport themes and exam-style traps. It is not a copied official past-question database.

## Suggested next improvements

- Add more Japanese-first explanations for learners taking the exam in Japanese.
- Add a dedicated "Answer Review" screen that groups questions by concept.
- Add glossary mode for Japanese AI terms.
