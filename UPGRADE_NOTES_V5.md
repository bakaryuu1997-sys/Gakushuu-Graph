# V5 Readable Study-First Upgrade

This version changes the product from graph-first to study-first.

## Main improvements

- New three-column layout:
  - Left: Study Navigation
  - Center: Lesson Content / Study views
  - Right: Focused Relation Graph
- Daily 10 Nodes moved into the main learning area.
- Added All Nodes view as a complete study index.
- Added Study Path view with ordered learning phases.
- Lesson content is now the central focus instead of the side panel.
- Right graph is now a supporting focused graph, not the main reading surface.
- Added Exam Focus view with important nodes and mini quiz.
- Favorites and Recent are now proper views.

## Why this is better

Large knowledge graphs are difficult to learn from when too many nodes are visible at once. This version prioritizes the lesson, learning order, daily tasks, and exam-focused review, while keeping the graph as a visual support tool.
