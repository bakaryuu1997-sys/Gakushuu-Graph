# AI Agent Rules

- Read README.md and source structure before editing.
- Do not add backend, login, database, or AI API without explicit confirmation.
- Keep data outside UI components.
- Keep components focused on rendering.
- Put logic into hooks or utilities.
- Avoid files over 200 lines when possible.
- Use strict TypeScript. No `any`.
- Do not refactor outside the requested scope.
- After edits, report changed files and validation steps.
